The TypeScript Compiler

In this section I learn how to configure and use the compiler for TypeScript.

Using watch mode

To compile a TypeScript file we run tsc [name of file].ts. This generates a new .js file of the same name. However, this can be tidious to do every time we change our TypeScript file. For that we add a flag --watch to the command which will run the compiler in watch mode. We can add this to the package.json file like so: 

  1. {
  2.   "name": "typescript-lessons",
  3.   "version": "1.0.0",
  4.   "description": "Sample typescript app",
  5.   "main": "app.js",
  6.   "scripts": {
  7.     "serve": "live-server --port=8090 --quiet",
  8.     "compile:type": "tsc app.ts --watch"
  9.   },
  10.   "author": "Peter Kitonga",
  11.   "license": "ISC",
  12.   "devDependencies": {
  13.     "live-server": "^1.2.1"
  14.   }
  15. }
Compiling entire projects

The script in the above section makes it easier to watch for changes in a specific file. This however will now work well for entire projects with multiple TypeScript files. To solve this we initialize a TypeScript configuration file via the command tsc --init. This generates a file in the project root folder called tsconfig.json. This inturn allows us to just run tsc --watch which will watch for changes made in any TypeScript file in the same project folder. This means we can modify our package.json file to: 

  1. {
  2.   "name": "typescript-lessons",
  3.   "version": "1.0.0",
  4.   "description": "Sample typescript app",
  5.   "main": "app.js",
  6.   "scripts": {
  7.     "serve": "live-server --port=8090 --quiet",
  8.     "compile:type": "tsc --watch"
  9.   },
  10.   "author": "Peter Kitonga",
  11.   "license": "ISC",
  12.   "devDependencies": {
  13.     "live-server": "^1.2.1"
  14.   }
  15. }
Including and Excluding files

There are a number of configurations that can be added to the tsconfig.json. One of the configurations is including and excluding files where we tell TypeScript to exclude compilation of the listed files and include only the listed files for compilation. 

  1. {
  2.   "compilerOptions": {
  3.     // ...compile configurations here
  4.   },
  5.   "exclude": [
  6.     "node_modules"
  7.   ],
  8.   "include": [
  9.     "app.ts"
  10.   ]
  11. }

Try it

To run the app, in your terminal run the following commands: 

  1. cd Lesson\ 02\ -\ TypeScript\ Compiler/app/
  2.  
  3. npm install // if you have not installed dependencies
  4.  
  5. npm run compile:type // compiles typescript to common js
  6.  
  7. npm run serve // starts application
