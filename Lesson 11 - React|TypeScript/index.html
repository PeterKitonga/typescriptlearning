React.js & TypeScript

In this section I learned how to use TypeScript in React.js. React.js is a very popular frontend framework with tones of features and configurability. It uses a component based for creating Single Page Applications(SPAs). Alternatives to React.js are Angular, Vue, Ember, Svelte.

Type assignment for Function Components

In React, TypeScript offers type safety which can be used to assist us during development. In this case we want to tell TypeScript that our component returns a function by infering it to the React.FC type. 

  1. import React from 'react';
  2.  
  3. const App: React.FC = () => {
  4.   return (
  5.     <div className="App">
  6.     </div>
  7.   );
  8. }
  9.  
  10. export default App;
Defining interfaces for props

In React, props are used as a store of value accross different components. They pass data from a parent component to its child components which can be retrieved within the child component for manipulation and rendering. For this to work we need to tell TypeScript what to expect as a prop and its types. For example, below we are telling TypeScript that our component TodoList expects a prop called items which is an array of objects. This is then passed as a generic type to the function component. 

  1. import React from "react";
  2.  
  3. interface TodoListProps {
  4.     items: { _id: string, text: string }[];
  5. }
  6.  
  7. const TodoList: React.FC<TodoListProps> = props => {
  8.     return (
  9.         <ul>
  10.             {props.items.map(item => <li key={item._id}>{item.text}</li>)}
  11.         </ul>
  12.     );
  13. };
  14.  
  15. export default TodoList;
Try it

To run the app, in your terminal run the following commands: 

  1. cd Lesson\ 11\ -\ React\|TypeScript/app/
  2.  
  3. npm install // if you have not installed dependencies
  4.  
  5. npm start // starts application
