Code

Here, I created a simple calculator that is triggered via an event listener on the Add! button below. Below is the TypeScript code for the simple calculator. 

  1. const button = document.getElementById("calculator-add-button");
  2. const input1 = document.getElementById("num1")! as HTMLInputElement;
  3. const input2 = document.getElementById("num2")! as HTMLInputElement;
  4.  
  5. const add = (num1: number, num2: number) => {
  6.     return num1 + num2;
  7. }
  8.  
  9. button.addEventListener("click", () => {
  10.     const calculator_result_span = document.getElementById("calculator-result-span");
  11.     calculator_result_span.innerText = `RESULT: ${add(+input1.value, +input2.value)}`;
  12. });

Result: 3

To run the .ts file we need to compile it via the command:

  1. tsc assets/scripts/first-app.ts
